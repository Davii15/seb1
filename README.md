# Kagumo Downloader Site

A professional download site for SEB Browser, built for Kagumo Teachers College.

## Features

- **Responsive Design**: Works perfectly on all devices
- **Download Tracking**: Tracks downloads and clicks using localStorage
- **Professional UI**: Modern gradient design with institutional branding
- **Secure Downloads**: Direct download functionality for SEB Browser
- **Analytics**: Built-in Vercel Analytics integration

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000) in your browser

## File Structure

- `app/page.tsx` - Main homepage with download functionality
- `app/layout.tsx` - Root layout with metadata and fonts
- `components/ui/` - Reusable UI components
- `public/` - Static assets (place seb-browser.exe here)

## Deployment

Deploy easily to Vercel by connecting your GitHub repository.
